### 1️⃣ What is the difference between var, let, and const?
`var` হলো function scope, `let` আর `const` হলো block scope। `let` এর value পরে change করা যায়, `const` এর যায় না।


### 2️⃣ What is the spread operator (...)?
`...` দিয়ে array বা object এর সব element আলাদা করে বের করা যায়। যেমন: `[...arr1, 4, 5]`


### 3️⃣ What is the difference between map(), filter(), and forEach()?
`forEach()` শুধু loop চালায়। `map()` নতুন array return করে element modify করে। `filter()` condition মিললে সেই element নিয়ে নতুন array return করে।


### 4️⃣ What is an arrow function?
`=>` দিয়ে function ছোট করে লেখার পদ্ধতি। যেমন: `const add = (a, b) => a + b;`


### 5️⃣ What are template literals?
Backtick `` ` `` দিয়ে string লেখা যায় আর `${}` দিয়ে variable বসানো যায়। যেমন: `` `আমার নাম ${name}` ``
